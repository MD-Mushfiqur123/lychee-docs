🚀 Quick Start

Lychee requires Ollama to be installed and running. Lychee acts as a drop-in superset — everything Ollama does, Lychee does too, plus more.

Install

git clone https://github.com/MD-Mushfiqur123/lychee.git
cd lychee
go build -o lychee .
sudo mv lychee /usr/local/bin/   # macOS / Linux

Start the Server

lychee serve

Server runs on http://localhost:11434 by default (same port as Ollama).

Run Your First Model

lychee run gemma3

Make an API Call

curl http://localhost:11434/api/chat \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemma3",
    "messages": [{"role": "user", "content": "Hello, Lychee!"}]
  }'

⌨️ CLI Commands

Command Description
lychee serve Start the Lychee API server on port 11434
lychee run <model> Run a model in interactive chat mode
lychee run <model> --experimental Run in agent sandbox mode with tool execution (bash, web search, filesystem)
lychee hf search <query> Search HuggingFace Hub for GGUF models
lychee hf pull <repo> Download a GGUF model from HuggingFace with resume & SHA256 verification
lychee hf pull <repo> --list List available quantization variants for a model
lychee hf pull <repo> --quant q5_k_m Download a specific quantization variant
lychee compare <prompt> <m1> <m2> Side-by-side model benchmarking with real-time streaming metrics
lychee stats --tui Interactive terminal dashboard — models, VRAM, tokens/sec, context usage
lychee scan Hardware detection with personalized model sizing recommendations
lychee generate-client Generate ready-to-run API client code in Python / JS / Go / Rust

🔌 API Reference

All endpoints are served at http://localhost:11434.

Method Endpoint Description
POST /api/chat Chat completions with streaming, tools & multi-turn conversation
POST /api/generate Text generation with streaming, format & options
POST /v1/chat/completions OpenAI-compatible chat completions endpoint
POST /v1/messages Anthropic-compatible Messages API with streaming
POST /v1/responses OpenAI Responses API — structured responses with JSON Schema validation
POST /api/structured Structured output with schema validation & auto-retry on invalid JSON
POST /api/conversations Persistent conversation memory — create, list, resume, delete
GET /api/conversations/<id> Retrieve conversation details and message history
POST /api/routes Define virtual model routes with multi-backend load balancing
POST /api/compose Multi-model pipeline composition — chain models sequentially

Example: Anthropic Messages API

Drop-in replacement — point the official Anthropic SDK to Lychee:

curl http://localhost:11434/v1/messages \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemma3",
    "max_tokens": 1024,
    "messages": [{"role": "user", "content": "Explain quantum computing in one sentence."}]
  }'

Example: Structured Output with Auto-Retry

curl http://localhost:11434/api/structured \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemma3",
    "prompt": "Extract name and age: Jane Doe is 28.",
    "schema": {
      "type": "object",
      "properties": {
        "name": {"type": "string"},
        "age": {"type": "integer"}
      },
      "required": ["name", "age"]
    },
    "max_retries": 3
  }'

Example: Model Composer Pipeline

curl http://localhost:11434/api/compose \
  -H "Content-Type: application/json" \
  -d '{
    "input": "Life is like a box of chocolates.",
    "steps": [
      {
        "model": "gemma3",
        "prompt": "Translate to French: {{input}}"
      },
      {
        "model": "phi3",
        "prompt": "Explain this French phrase: {{step[0].output}}"
      }
    ],
    "stream": false
  }'

Example: Model Router Setup

curl http://localhost:11434/api/routes \
  -H "Content-Type: application/json" \
  -d '{
    "name": "fast-route",
    "endpoints": [
      {"host": "http://localhost:11434", "model": "gemma3", "weight": 2},
      {"host": "http://localhost:11435", "model": "phi3", "weight": 1}
    ],
    "strategy": "weighted_round_robin"
  }'

Python SDK Quick Example

from lychee import Lychee

client = Lychee(base_url="http://localhost:11434")

# Structured output with auto-retry
result = client.structured.generate(
    model="gemma3",
    prompt="Extract: Jane Doe, 28, engineer",
    schema={
        "type": "object",
        "properties": {
            "name": {"type": "string"},
            "age": {"type": "integer"},
            "occupation": {"type": "string"}
        }
    }
)
print(result.data)

JavaScript SDK Quick Example

import { Lychee } from 'lychee-js';

const client = new Lychee({ baseURL: 'http://localhost:11434' });

// Chat with memory
const conv = await client.conversations.create({
  model: 'gemma3',
  messages: [{ role: 'user', content: 'Hello Lychee!' }]
});

// Resume conversation
const reply = await client.chat.create({
  model: 'gemma3',
  conversationId: conv.id,
  messages: [{ role: 'user', content: 'What can you do?' }]
});
console.log(reply.message.content);

Anthropic SDK Compatibility (Drop-in)

from anthropic import Anthropic

client = Anthropic(
    base_url="http://localhost:11434/v1",
    api_key="lychee-local"
)

message = client.messages.create(
    model="gemma3",
    max_tokens=1024,
    messages=[{"role": "user", "content": "Hello!"}]
)
print(message.content[0].text)
